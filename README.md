# Hangman Game

This is a classic Hangman Game built with Python and Pygame. The player has to guess the hidden word by selecting letters. Each incorrect guess brings the player closer to losing the game. The project features a graphical user interface, word selection logic, and interactive gameplay.

## Features
- Interactive GUI using Pygame
- Random word selection
- Letter-based guessing system
- Win/Lose conditions
- Beginner-friendly Python project

## Technologies Used
- Python
- Pygame
